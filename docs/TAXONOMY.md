# 分类体系

## 考试范围

首版优先覆盖：

- 国考
- 四川省考
- 重庆省考
- 事业单位/事业编考试

字段建议：

- `exam_category`：`公务员`、`事业编`
- `exam_type`：`国考`、`省考`、`事业编`
- `region`：`全国`、`四川`、`重庆`，后续可扩展到其他省份
- `year`：四位年份
- `paper_kind`：`行测`、`申论`、`职测`、`综应`、`公基`

## 政策理论标签

基础标签：

- `时政热点`
- `党建`
- `经济`
- `法治`
- `民生`
- `生态`
- `基层治理`
- `科技教育人才`
- `文化建设`
- `国际观察`

标签规则：

- 每篇文章 1-3 个 `tags`，只标注文章主旨大类。
- `source` 不作为标签，单独写 frontmatter。
- `就业`、`社会保障`、`乡村振兴`、`应急管理`、`营商环境` 等细分主题写入 `topics`，不写入 `tags`。
- `申论素材`、`规范表述` 属于 `topics`，用于申论复习用途，不作为关系图谱主标签。
- 标签以复习、检索和关系图谱为目标，不追求新闻栏目完全还原。

## 政策理论 Topics

`topics` 用作二级索引，允许同时记录细分主题和复习用途：

- 细分主题：`就业`、`社会保障`、`乡村振兴`、`应急管理`、`营商环境`、`科技创新`、`人才培养`、`文化传承`、`基层减负`、`公共服务`
- 复习用途：`申论素材`、`规范表述`

## 题型

公务员行测：

- `言语理解`
- `数量关系`
- `判断推理`
- `资料分析`
- `常识判断`

公务员申论：

- `归纳概括`
- `综合分析`
- `提出对策`
- `贯彻执行`
- `文章写作`

事业编：

- `职业能力倾向测验`
- `综合应用能力`
- `公共基础知识`

`question_format` 用于记录题目形式：

- `单选`
- `多选`
- `判断`
- `不定项`
- `材料分析`
- `写作`
- `客观题`
- `主观题`

公基 `knowledge_points` 首版建议：

- `政治`
- `法律`
- `经济`
- `公文`
- `管理`
- `人文科技`
- `省情市情`

## 复核状态

题目默认进入 `needs_review`。全量精校后可改为 `verified`，要求题干、选项/作答要求、答案、解析、题型和知识点均已确认。无法确认答案或解析的题目标记为 `blocked` 或继续保留 `needs_review`，并在 `parse_warnings` 记录原因。

## 难度

首版使用三级：

- `easy`
- `medium`
- `hard`

没有足够证据时默认 `medium`，后续由作答正确率和耗时校正。
