# Classification Rules

- 行测题型：言语理解、数量关系、判断推理、资料分析、常识判断。
- 申论题型：归纳概括、综合分析、提出对策、贯彻执行、文章写作。
- Ambiguous objective questions default to `常识判断` only when no stronger textual signal exists.
- Material tables, ratios, growth rates, and charts usually imply `资料分析`.
- Argument, graph, definition, analogy, and spatial reasoning usually imply `判断推理`.
