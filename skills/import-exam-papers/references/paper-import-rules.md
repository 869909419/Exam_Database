# Paper Import Rules

- Scope is 国考, 四川省考, 重庆省考 first.
- Keep `exam_type`, `region`, `year`, `source_url`, `source_file`, `question_count`, and `import_status`.
- Do not bypass commercial sites, paywalls, logins, captchas, or anti-scraping controls.
- Prefer user-provided copyable PDFs or Markdown when public websites are unstable.
- Preserve the original file path in SQLite for later audit.
